# Logitech DFGT Active FFB Driver for Arduino Uno & USB Host Shield

Logitech Driving Force GT (DFGT) を **Arduino Uno R3 + USB Host Shield 2.0 (MAX3421E)** を用いて動作させ、自律物理演算による高度なフォースフィードバック (FFB) と、他デバイス間との超低遅延双方向シリアル通信を実現するためのプログラムです。

---

## 特徴

*   **900度・14-bitネイティブモードへの自動移行**
    *   起動時にDFGT本来の900度Mode・14-bit（0〜16384）ステアリング信号を出力する「ネイティブモード」へ自動で切り替えます。
      
*   **自律物理演算による FFB 合成エンジン**
    *   以下の4つの要素をArduinoがリアルタイムに高速演算・合算し、FFBとして出力します
        1.  **スプリング効果：** センターへ引き戻す復元力。
        2.  **ダンパー効果（粘り抵抗）：** ハンドルを回す速度に応じた、実車の重ステ（停車時）のような油圧的粘り重さ。
        3.  **バイブレーション：** 路面の荒れ等を表現する継続的なガタガタ微振動。
        4.  **単発衝撃：** 壁への衝突やシフトショックなどの瞬間的な「ゴン！」という打撃衝撃。
           
*   **Arduinoリセット時の「キャリブレーション暴走防止」**
    *   Arduinoにプログラムを書き込んだり、リセットボタン（ホットリセット）を押したりした際、DFGTがすでにネイティブモードとして起動・キャリブレーション完了しているかをUSBのVID/PIDから自動判定。多重リセットによるハンドルの回転途中のフリーズやズレを徹底的に防止します。
    *   
*   **超低遅延・双方向シリアルCSV通信**
    *   **送信：** DFGTのパケット受信と完全に同期し、初期状態では、アナログ3軸（ステアリング、アクセル、ブレーキ）のみを最速でシリアル送信します。
      (すべてのボタンに対応した関数を作成したのでその関数を呼び出せばこちらも出力が可能です)
    *   **受信：** 他のArduino等から、バネ強度・ダンパー強度・振動・単発衝撃コマンドをCSV形式で受信可能。

---

## 🛠️ 必要なハードウェア

1.  **Arduino Uno R3** (または互換機)
2.  **USB Host Shield 2.0** (MAX3421Eチップ搭載)
3.  **Logitech Driving Force GT (DFGT)** 本体 ＆ 付属ACアダプタ
4.  **Uno用外部電源：** 9V〜12V (1A以上) のACアダプタ
    *   *※PCのUSB給電のみでは、DFGT接続時の突発的な電力不足により、USBホストシールドが通信ハングを起こしたり、Arduinoがブラウンアウトリセット（強制再起動）される原因となります。DCジャックからの外部給電を強く推奨します。(USB電源でデバックしていましたがかなり不安定)*

---

## ファイル

*   `DFGT_Arduino_Host.ino` ：メインスケッチ。タイムライン接続制御、FFB演算、シリアル送受信を処理します。
*   `hidjoystickrptparser.cpp` ：USBパケット受信部。ボタンやペダルなどの生ビットを物理変数へデコードします。
*   `hidjoystickrptparser.h` ：DFGT用のネイティブ12バイトパケット構造体、および各変数の定義。

##  Arduino等との接続方法

もう1台の制御用Arduino等とシリアル通信（115200bps）で相互通信を行う場合、HardWareSerialを使用します。
処理がR3ではギリギリなので、外部Arduinoを接続し、処理を分割することを推奨します。
```text
 [ 本機 (FFB/Host側) ]                      [ 相手機 (メイン制御等) ]
     RX (0番ピン)   <--------------------->   TX (送信)
     TX (1番ピン)   <--------------------->   RX (受信)
         GND       <--------------------->   GND (共通グラウンド)

(ACアダプタ動作時Vin端子どうしを接続するとアダプタが一つで済むので便利です、USB書き込み時アダプタを抜くことを
徹底してください！！！電源がUSBに逆流する可能性があります！！！)
```
